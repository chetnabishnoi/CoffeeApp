<h1>CoffeeApp</h1>
<p>Shows a list of coffee shops and a small description.</p>

<p>It is based on the react-native and makes use of the following</p>

<li>react-redux </li>
<li>redux saga</li>
<li>redux bees</li>
<li>react native maps</li>

<h2>ToRun</h2>
  <p>npm install</p>
  <p>npm start</p>
  <p>react-native run-ios or react-native run-android</p>
